# 欢迎来到我的博客

这个博客使用 VitePress 构建，文章以 Markdown 文件的形式存储在 GitHub Repository 中。

后续你可以通过 Pages CMS 在浏览器里新增、编辑、删除文章，也可以上传图片。每一次内容变更都会提交到 GitHub，并由 GitHub Actions 自动部署到 GitHub Pages。

## 写作工作流

1. 登录 Pages CMS。
2. 新建一篇 Markdown 文章。
3. 填写标题、摘要、发布日期、分类和标签。
4. 保存后等待 GitHub Actions 自动发布。

## 支持的内容

文章可以包含代码块、表格、Mermaid 图表和数学公式。

```ts
const blog = 'VitePress + Pages CMS + GitHub Pages'
console.log(blog)
```

```mermaid
flowchart LR
  Write[写文章] --> Commit[提交到 GitHub]
  Commit --> Build[Actions 构建]
  Build --> Publish[发布到 Pages]
```

行内公式示例：$E = mc^2$。
